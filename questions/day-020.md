# SQL Scenario-Based Interview Question: SaaS Consecutive High Usage Identification

## Business Scenario
A SaaS company offers a cloud storage platform with tiered pricing. While customers pay a base subscription, any storage usage beyond a certain free allowance (e.g., 500 GB per month) is billed additionally. The Sales and Marketing teams want to identify "Power Users" – customers who consistently exceed their free 500 GB storage allowance for a sustained period. These customers are prime candidates for an upsell to higher-tier enterprise plans or could indicate a need for a review of standard free allowances. As a Data Analyst, you need to provide a list of these power users.

Assume:
```sql
CURRENT_DATE = '2023-08-01'
```

---

# Database Setup

## Create Customers Table
```sql
CREATE TABLE customers (
    customer_id INTEGER PRIMARY KEY,
    customer_name VARCHAR(100) NOT NULL
);
```

## Insert Customers Data
```sql
INSERT INTO customers (customer_id, customer_name) VALUES
(101, 'Alice Smith'),
(102, 'Bob Johnson'),
(103, 'Charlie Brown'),
(104, 'David Lee'),
(105, 'Eve Davis');
```

## Create Monthly_Usage Table
```sql
CREATE TABLE monthly_usage (
    usage_id SERIAL PRIMARY KEY,
    customer_id INTEGER NOT NULL,
    usage_month DATE NOT NULL, -- Stored as the first day of the month
    storage_gb_used INTEGER NOT NULL,
    FOREIGN KEY (customer_id) REFERENCES customers(customer_id)
);
```
