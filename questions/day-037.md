# SQL Scenario-Based Interview Question: Campaign Attributed First Purchases

## Business Scenario
An E-commerce company is heavily investing in "Reactivation" marketing campaigns to re-engage dormant customers. As a Data Analyst, your team needs to measure the direct impact of these campaigns. Specifically, the marketing team wants to identify the *first purchase* a customer makes after interacting with a specific 'Reactivation' campaign, but only if that purchase occurs within a 7-day window from the interaction timestamp. This helps them understand which individual campaign interactions effectively prompt an immediate return to purchase.

Assume:
```sql
CURRENT_DATE = '2023-10-26'
```

---

# Database Setup

## Create Customers Table
```sql
CREATE TABLE Customers (
    customer_id INT PRIMARY KEY,
    customer_name VARCHAR(100),
    email VARCHAR(100)
);
```

## Insert Customers Data
```sql
INSERT INTO Customers (customer_id, customer_name, email) VALUES

(4, 'David Beckham', 'david@example.com'),

```

## Create Campaigns Table
```sql
CREATE TABLE Campaigns (
    campaign_id INT PRIMARY KEY,
    campaign_name VARCHAR(100),
    campaign_type VARCHAR(50) -- e.g., 'Reactivation', 'Promotional', 'Awareness'
);
```

## Insert Campaigns Data
```sql

(101, 'Reactivation_A_Q4', 'Reactivation'),
(102, 'Reactivation_B_Flash', 'Reactivation'),
(103, 'Promotional_Holiday', 'Promotional'),
(104, 'Reactivation_C_Loyalty', 'Reactivation');
```

## Create Campaign_Interactions Table
```sql
CREATE TABLE Campaign_Interactions (
    interaction_id INT PRIMARY KEY,
    customer_id INT,
    campaign_id INT,
    interaction_timestamp TIMESTAMP,
    FOREIGN KEY (customer_id) REFERENCES Customers(customer_id),
    FOREIGN KEY (campaign_id) REFERENCES Campaigns(campaign_id)
);
```

# Task
Write a SQL query to identify the *first purchase* made by a customer after interacting with a 'Reactivation' campaign. The purchase must occur within 7 days of the campaign `interaction_timestamp`. Each successful `interaction_id` should ideally link to at most one `order_id` (the first qualifying one).

The output should include the `campaign_name`, `interaction_id`, `customer_id`, `order_id`, `order_timestamp`, and `total_amount` for these attributed first purchases.
