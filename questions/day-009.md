# SQL Scenario-Based Interview Question: Identify Disengaged Campaign Participants

## Business Scenario
You are a Data Analyst at a leading E-commerce company. The marketing team recently ran a "Fall Fashion Frenzy" email campaign promoting specific categories. They want to identify customers who showed initial interest by clicking on the campaign link but *did not* follow through with a purchase of products from the promoted category within a critical follow-up period. This segment of customers needs to be targeted with a re-engagement strategy.

Assume:
```sql
CURRENT_DATE = '2023-10-26'
```

---

# Database Setup

## Create Customers Table
```sql
CREATE TABLE Customers (
    customer_id INT PRIMARY KEY,
    customer_name VARCHAR(100),
    email VARCHAR(100) UNIQUE
);
```

## Insert Customers Data
```sql
INSERT INTO Customers (customer_id, customer_name, email) VALUES
(101, 'John Doe', 'john.doe@example.com'),
(102, 'Jane Smith', 'jane.smith@example.com'),
(103, 'Bob Johnson', 'bob.j@example.com'),
(104, 'Alice Williams', 'alice.w@example.com'),
(105, 'Charlie Brown', 'charlie.b@example.com');
```

## Create Products Table
```sql
CREATE TABLE Products (
    product_id INT PRIMARY KEY,
    product_name VARCHAR(100),
    category VARCHAR(50),
    price DECIMAL(10, 2)
);
```

## Insert Products Data
```sql
INSERT INTO Products (product_id, product_name, category, price) VALUES
(1, 'Wool Scarf', 'Fall Apparel', 25.00),
(2, 'Leather Boots', 'Fall Apparel', 120.00),
(3, 'Denim Jacket', 'Casual Wear', 75.00),
(4, 'Silk Blouse', 'Formal Wear', 60.00),
(5, 'Knit Sweater', 'Fall Apparel', 55.00),
(6, 'Running Shoes', 'Sportswear', 90.00);
```

## Create Marketing_Campaigns Table
```sql
CREATE TABLE Marketing_Campaigns (
    campaign_id INT PRIMARY KEY,
    campaign_name VARCHAR(100),
    start_date DATE,
    end_date DATE,
    promotion_category VARCHAR(50) -- The category of products promoted by the campaign
);
```

## Insert Marketing_Campaigns Data
```sql
INSERT INTO Marketing_Campaigns (campaign_id, campaign_name, start_date, end_date, promotion_category) VALUES
(1, 'Fall Fashion Frenzy', '2023-10-01', '2023-10-15', 'Fall Apparel'),
(2, 'Summer Sale', '2023-07-01', '2023-07-15', 'Sportswear');
```

## Create Campaign_Interactions Table
```sql
CREATE TABLE Campaign_Interactions (
    interaction_id INT PRIMARY KEY,
    customer_id INT,
    campaign_id INT,
    interaction_type VARCHAR(20), -- e.g., 'open', 'click', 'view'
    interaction_date DATE,
    FOREIGN KEY (customer_id) REFERENCES Customers(customer_id),
    FOREIGN KEY (campaign_id) REFERENCES Marketing_Campaigns(campaign_id)
);
```

## Insert Campaign_Interactions Data
```sql
INSERT INTO Campaign_Interactions (interaction_id, customer_id, campaign_id, interaction_type, interaction_date) VALUES

(1002, 102, 1, 'click', '2023-10-03'), -- Jane clicked Fall Fashion

```

## Create Orders Table
```sql
CREATE TABLE Orders (
    order_id INT PRIMARY KEY,
    customer_id INT,
    order_date DATE,
    total_amount DECIMAL(10, 2),
    FOREIGN KEY (customer_id) REFERENCES Customers(customer_id)
);
```

## Insert Orders Data
```sql
INSERT INTO Orders (order_id, customer_id, order_date, total_amount) VALUES
(1, 102, '2023-10-04', 120.00), -- Jane bought soon after clicking
(2, 103, '2023-10-06', 75.00),  -- Bob bought something, but not from the promoted category
(3, 104, '2023-10-20', 55.00),  -- Alice bought later than 14 days
(4, 101, '2023-10-25', 60.00);  -- John bought something later, but not from the promoted category
```

## Create Order_Items Table
```sql
CREATE TABLE Order_Items (
    order_item_id INT PRIMARY KEY,
    order_id INT,
    product_id INT,
    quantity INT,
    item_price DECIMAL(10, 2),
    FOREIGN KEY (order_id) REFERENCES Orders(order_id),
    FOREIGN KEY (product_id) REFERENCES Products(product_id)
);
```

## Insert Order_Items Data
```sql
INSERT INTO Order_Items (order_item_id, order_id, product_id, quantity, item_price) VALUES

```

# Task
Identify all customers who:
1.  Clicked on the "Fall Fashion Frenzy" campaign (campaign_id = 1).
2.  Did NOT make any purchase of a product categorized as 'Fall Apparel' within 14 days of their *specific click interaction date* for that campaign.
3.  The analysis should consider only "click" interactions for the specified campaign.

Return the `customer_id`, `customer_name`, `email`, `campaign_name`, and the `interaction_date` of their click.
